Instructions

To Write A Tag

1. Go to the Set Tag Details tab

2. Select a pre-set or user-set tag through the dropdown menu

3. Go to The Write Tag Tab

4. Select 'Start Writing To Tags'

5. The application will listen for any tags that are near the phone

6. Place the tag to write to next to the phone for around 1-3 seconds

7. A pop up will appear alerting you when the tag has been successfully written

8. The "Live Tag Data" section is a readout of the GUID that has been written to the tag.

9. Repeat or select 'Stop Writing To Tags' when finished writing

Add New Material

1. Go to the Set Tag Details tab

2. Select 'New Material'

3. An input box to enter the material's name will appear

4. Enter the custom material's name and press OK.

4a. Example Name: pla-natural

5. An input box to enter for the material's GUID will appear

5a. Enter the custom material's GUID

6. The custom material will be added to the dropdown menu and will be immediately writable upon selection
